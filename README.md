# yuriQt

> 这是我本人的简单qt项目，是对简单qt控件的整理应用
>
> 本项目 基于 `Qt6` 进行开发

## 实现功能

1. MySQL 数据库登陆
2. c/s 通信测试
3. mysql 数据查询
4. 便签功能
